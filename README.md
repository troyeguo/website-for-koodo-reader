### 技术栈

使用 **Next.js** 实现服务端渲染，使用 **Bootstrap** 实现响应式，使用 **next-i18next** 实现国际化，使用 **aos** 实现页面动画

### 网站地址

[点我前往](https://koodo.960960.xyz)

### 运行源码

```bash
git clone https://github.com/troyeguo/website-for-koodo-reader.git
cd website-for-koodo-reader
yarn
yarn dev
```
